# AhmedGPT Ahmed Sayed — OpenRouter Edition

AhmedGPT Ahmed Sayed is a simple terminal-based AI assistant that allows users to communicate with multiple AI models through OpenRouter.

The project focuses on:

- speed
- simplicity
- terminal usability
- multi-model support

---

## ⚡ Features

- 🔗 OpenRouter API integration
- 🌍 Automatic language detection
- 🗣️ Multi-language support
- 💬 Interactive chat session
- 🎨 Stylish CLI interface
- 🔧 Custom model configuration
- 💾 Auto-saved configuration file

---

## 📦 Requirements

Before running the project make sure you have:

- python 3.6+
- pip installed
- OpenRouter API key

---

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/FBI0NINJA/wormgpt
cd worm-gpt
```

Run the program

```bash
python wormgpt.py
```

---

## 🔑 API Key Setup

You can set your API key from the menu

```
[3] Set API Key
```

Or manually edit the configuration file:

```json
{
  "api_key": "YOUR_API_KEY",
  "base_url": "https://openrouter.ai/api/v1",
  "model": "deepseek/deepseek-chat",
  "language": "English"
}
```

---

## 🧠 Usage

Run the program

```bash
python wormgpt.py
```

Main menu

```
[ Main Menu ]

1. Language
2. Model
3. Set API Key
4. Start Chat
5. Exit
```
